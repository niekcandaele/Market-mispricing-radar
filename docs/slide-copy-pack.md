# Slide Copy Pack

## Purpose

This is the paste-ready on-slide text layer for the submission deck.

Use it when building Google Slides so the deck can be assembled quickly without rewriting headlines and bullets from scratch.

## Style rules

- keep each slide concise
- prefer one clear headline plus 2 to 4 bullets
- do not overclaim
- let the screenshot do visual work when one is present
